

## 4. `Hotel-WebSite`

# Hotel Website

Bu proje bir otel web sitesi taslağıdır ve otelin tanıtımı için kullanılabilir bir şablon sunar.

## Özellikler

- Anasayfa, Odalar, Hizmetler, Galeri, Rezervasyon ve İletişim sayfaları  
- Otel odalarının kartlarla gösterimi  
- Rezervasyon formu (statik form ile)  
- Responsive tasarım (mobil ve masaüstü uyumlu)

## Teknolojiler

- HTML  
- CSS
